# Device Settings

Device settings are settings that are applied per device. There are values that are applied as properties of the device.

```js
node.property = value; // Example changing a setting on a node
```

### `quit_program_on_device_disconnect` (bool [default:true]) Sets whether the program should forcefully quit when the device disconnects.
